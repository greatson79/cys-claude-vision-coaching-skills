## 4. 산출물

### 벤치마크 ② 원본 메타데이터 (엔진 list_benchmarks)
- 이름: 2016~2035 미래산업 미래지도 (박사님 2016 작성)
- 원본 시간축: 2016~2035 (20년), 3구간: 2016~2020 / 2020~2030 / 2030~2035
- 원본 영역: IT·BT·NT·RT·ST + 사회·정치
- 원본 핵심 사건: 1차 3D프린팅 → 인공지능 혁명 → 2차 가상혁명 → 매트릭스·환상사회 → 4차 산업혁명

### 현재 기준 재설정 (엔진 compute_intervals 2026)
- 단기 2026~2031: ST(양자센싱) 부상, 양자 SDK 보급, NISQ 머신 클라우드
- 중기 2031~2041: 양자 오류정정 1차 돌파, 양자-AI 융합, BT(양자약물 시뮬레이션)
- 장기 2041~2056: 폴트톨러런트 양자컴퓨터 상용화, 글로벌 양자 인터넷, RSA 암호 폐기

### 영역 교체 (원본 IT·BT·NT·RT·ST → 양자컴퓨터 분야 매핑, Claude 코칭)
| 원본 영역 | 양자컴퓨터 분야 적용 |
|---|---|
| IT | 양자 하드웨어(초전도·이온트랩·중성원자·포토닉) |
| BT | 양자 약물 시뮬레이션·단백질 폴딩 |
| NT | 양자 소재·반도체 양자점 |
| RT | 양자 센서 로보틱스 |
| ST | 양자센싱·우주 양자 통신 |
| 사회 | STEM 인력·국가 양자 전략 |
| 정치 | 미·중 양자 패권·수출통제 |

### 4가지 고려사항
- ① 직면: NISQ→FTQC 전환, 양자 인력 부족, 미국 CHIPS·중국 양자굴기
- ② 필요: 양자SW 인력, K-양자 정책 협력, 글로벌 표준 참여
- ③ 약한 신호 5소스: arXiv quant-ph, IBM/Google/IonQ 로드맵, 양자 스타트업 특허, 양자 학회 발표, 양자 안전 암호 표준화 단신
- ④ Wildcard: 비약적 진보 = 양자 우월성 입증된 산업 응용 등장 / 붕괴 = 양자 거품 붕괴·국가 간 양자 봉쇄
