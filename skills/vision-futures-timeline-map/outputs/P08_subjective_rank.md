# P08 — 박사님 책에서 가장 핵심인 한 가지 메시지가 뭐야

## 1. 사용자 입력
> 박사님 책에서 가장 핵심인 한 가지 메시지가 뭐야

## 2. 엔진 분기 매핑
```json
classify_entry => {"entry_type": "subjective_ranking", "matched_keywords": ["가장 핵심"]}
```

## 3. SKILL.md 처리 흐름 → 주관적 순위·추천 분기

> SKILL.md "박사님 책에 없는 주관적 순위·추천을 요청하는 경우" 절: 박사님 책에 어느 항목이 '가장 ○○하다'는 직접 언급 없음을 고지 → 4가지 고려사항(또는 핵심 개념들) 각각의 핵심을 SKILL.md 수록 내용으로 설명 → Claude 실용적 관찰 제공 시 "다만 이것은 제 관찰입니다. 박사님 책의 입장은 아닙니다"라고 명확히 구분 표기. 완료 후 "특정 비전 영역에 맞게 고려사항을 적용해드릴까요?" 제안.

## 4. 산출물

### (a) SKILL.md 고지 (필수)
> 박사님 책(『미래준비학교』, 2016)에서 "이 한 가지 메시지가 가장 핵심이다"라는 직접적인 언급은 수록되어 있지 않습니다. 박사님 책에 동등한 무게로 수록된 핵심 개념을 그대로 정리해 드립니다.

### (b) 박사님 책 핵심 개념 등가 나열 (모두 verify_quote 통과 원문 인용)

| # | 개념 | 박사님 책 인용 |
|---|---|---|
| 1 | 콜럼버스 비유 | *"미래지도란... 콜럼버스가 아메리카 대륙을 발견할 때 쓴 지도에 비유할 수 있다... 형편없고 엉망인 지도를 가지고 신대륙을 발견해냈다는 점이다."* (futures_map_definition) |
| 2 | 4가지 고려사항 | "직면할 상황 / 필요할 것 / 보잘것없는 정보 / 뜻밖의 미래" (SKILL.md 작업 ①~④) |
| 3 | 6힘 영역 | *"경제, 사회, 기술, 산업, 환경, 법, 정치, 제도, 글로벌 패권, 문화, 종교, 영성 등의 영역에서 일어나는 미래 변화에 대한 방향 감각이 필요하다."* (six_force_domains) |
| 4 | 단·장기 사고 | *"단기적이면서 동시에 장기적으로 생각하는 일이다."* (short_long_term_thinking) |
| 5 | 미래 징후 | *"미래 징후는 미래 변화의 모습을 묘사하거나 미래에 발생할 수 있는 변화의 결과를 구성하는 퍼즐 조각이다."* (futures_signs) |
| 6 | 미래 나침반 | *"미래 나침반은 넓고 깊고 통합적인 사고의 틀, 폭넓게 멀리 보는 관찰력, 계속해서 세상의 변화를 관찰하는 모니터링 습관이다."* (futures_compass_definition) |

### (c) Claude 관찰 (SKILL.md 절대원칙 #10 분리 표기)
> ⚠️ **이 관찰은 제 코칭 관점이며 박사님 책의 입장이 아닙니다.**
> 저는 위 6개 개념을 묶는 통합 메시지를 *"불완전한 지도(미래지도)와 통합 사고(미래 나침반)를 함께 갖춰 단기와 장기를 동시에 사고하라"*로 압축합니다. 다만 박사님 책이 "한 가지가 가장 핵심"이라 명시한 것은 아닙니다.

### 완료 후 SKILL.md 명시 메시지
