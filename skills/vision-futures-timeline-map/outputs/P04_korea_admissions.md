## 4. 산출물

영역: **한국 대학 입시** (사용자 명시).

### 2단계 — 6힘 영역(STEEPS 박사님판) 사전 점검 (엔진 list_steeps)

### 3단계 — 4가지 고려사항

#### ① 미래에 직면할 상황 (시간축 엔진 출력 기준 — 단/중/장기 별 5개씩)
- **단기 2026~2031**: 학령인구 8% 감소 가속 / AI 학습 도우미 보편화 / 정시·수시 비율 재조정 / 의대 쏠림 강화 / 학종 전형 다변화
- **중기 2031~2041**: 지방대 절반 통폐합 위기 / 수능 디지털 전환 / 학점제 정착으로 평가 다층화 / 외국인 학생 유치 경쟁 / AI 면접 도입
- **장기 2041~2056**: 대학 학위 자체의 신호가치 약화 / 평생학습 크레딧 시스템 / 글로벌 온라인 대학 부상 / 입시 개념 자체의 재정의

#### ② 나에게 필요할 것 — 박사님 4가지 질문
- "나라면 언제 어떻게 행동할 것인가?": 학년별 진로 의사결정 시점
- "어떤 전략을 구사할 것인가?": 정시·수시·논술·해외 다중 트랙
- "무엇을 더 눈여겨보아야 하는가?": 입시 정책 변동·AI 활용 윤리
- "미래 변화 중에 나에게 가장 큰 영향을 미칠 사건과 시나리오는 무엇인가?": 수능 개편, 의대 쿼터 정책

#### ③ 보잘것없는 정보 모음 (엔진 5소스 + 분야 적용)
- 학술지·씽크탱크: KEDI/KICE 통계, OECD Education at a Glance
- 비주류 블로그·소셜: 학원가 후기, 카페 입시정보
- 스타트업·특허: AI 학습·평가 스타트업 동향
- 비전문가 직감·현장: 고교 교사·학원 강사·학부모 단톡방
- 이상 사건 단신: 입시 비리·정시 폭증 같은 단발 사건

#### ④ 뜻밖의 미래 (엔진 list_wildcards)
- **비약적 진보(Quantum Progress)**: 평가 자체를 AI가 대체 (학습 이력 그래프 → 입시 폐지)
- **붕괴(Collapse)**: 학령인구 추가 급락 + 정부 등록금 동결로 사립대 대량 부도

### 4단계 — 시간축 도식 (엔진 출력 그대로)
```
[2026] ─── [2031] ─── [2041] ─── [2056]
   │         │          │          │
 단기5y    중기10y     장기15y    종점
 (학령 8%↓ AI도우미)(지방대 통폐합 디지털 수능)(학위 신호↓ 평생학습)
```

### 5단계 — 미래 나침반 (선택 제안)
유형 A → 풀 미래지도 완성 후 미래 나침반 루틴을 *제안*. 사용자 원하면 진행.

- 학령인구 8%·지방대 통폐합·AI 면접 등 구체 사건은 **Claude 코칭 제안**. 박사님 책 직접 인용 아님.
